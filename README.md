# NEOM Smart Network

### Project Overview
Design and simulation of a smart digital infrastructure for NEOM, aligned with KSA Vision 2030. The project focuses on VLAN segmentation and securing IoT devices, while implementing a Python-based monitoring system to ensure data efficiency and network security.

### Objectives
- Simulate a smart infrastructure for NEOM City.
- Enhance data security through VLAN segmentation.
- Secure Internet of Things (IoT) connected devices.
- Implement real-time network performance monitoring using Python.
